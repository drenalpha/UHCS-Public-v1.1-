
# UHCS-Public v1.1 — Anti-Hallucination Prompt (Open Version)

**Tagline:**  
Demonstrates robust anti-hallucination principles safely for public use.

---

## Overview

UHCS-Public v1.1 is an AI prompt designed to reduce hallucinations and ensure answers are based **only on user-provided source material**.  

This version is intended for **educational, research, and demonstration purposes** and is safe to share publicly. It showcases anti-hallucination principles without exposing proprietary UHCS-2.0 features.

---

## Key Features

- Uses only **user-provided source material** for answers  
- Explicitly flags **unsupported or ambiguous claims**  
- Encourages **traceable citations** for transparency  
- Clear output format with readable paragraphs or bullet points  
- Demonstrates anti-hallucination best practices for students, researchers, and educators  

**Proprietary Features Retained Privately:**  
UHCS-2.0’s triple-layer pipeline, optional modes, confidence scoring, and persistent session logic are **not included**.

---

## Why This Matters

AI hallucinations are a common problem when generating answers without verified sources. UHCS-Public v1.1 demonstrates **safe source-only reasoning** principles and transparency in outputs, making it ideal for:

- Students learning source-based reasoning  
- Researchers needing verifiable AI outputs  
- Educators demonstrating safe AI usage in classrooms
